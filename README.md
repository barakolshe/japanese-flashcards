# Japanese Flashcards

A simple, personal web app for learning Japanese with flashcards.

Upload a CSV file of words and the app turns them into a deck of flashcards. Flip each card to reveal the meaning, mark whether you got it right, and shuffle the deck to keep practicing. When you want, export your cards back out to a CSV file so you can save and organize your own word lists.

There's no sign-up and nothing to set up — just open it, load your words, and start studying.
